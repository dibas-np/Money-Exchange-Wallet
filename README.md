
# Money Exchange Wallet

Money exchange wallet is a frontend development of flutter application.

# Pages

* Splash 
* Welcome
* Location Access
* Login
* Signup
* Home
* Profile
* Cards Collection
* Send Money
* Withdraw Money
* Exchange Money
* Deposit Money
* Scan to pay
 

## Built With

* [Flutter](https://flutter.dev/)
* [Dart](https://dart.dev/)
